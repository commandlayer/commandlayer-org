Lookup

Resolve verbs to schemas and agent-cards.

Tooling can use this endpoint to map a verb name to its canonical request and receipt schemas, plus a reference agent-card.

Static registry

A static JSON document is served at /lookup/verbs.json: 

GET https://commandlayer.org/lookup/verbs.json

This registry can be generated from the underlying schema and agent-card manifests and kept in sync with each release.
