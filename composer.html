What this demo proves

The Commons layer is a real, usable contract — not just words in a README.

  • You pick the verb; the shapes stay stable.
  • Every call returns a typed receipt.
  • Trace IDs thread through calls for multi-agent workflows.
  • It’s easy to swap this mock runtime for a real x402/8004 stack while keeping the same request/receipt contracts.
