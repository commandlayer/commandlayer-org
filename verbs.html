How to read this catalog

Each verb represents a unit of machine intent that can be bound to a schema and an Agent Card:

  • Type: Commons (protocol public goods) or Commercial (economic flows).
  • Category: AI, commerce, operations, finance, comms, or other.
  • Status: Live, in flight, or reserved.
  • ENS: the semantic handle (e.g. summarizeagent.eth).
