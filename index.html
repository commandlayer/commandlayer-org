Canonical verbs, schemas & discovery for AI agents.

CommandLayer v1.0.0 defines a neutral naming and schema layer so builders can create their own agents that interoperate across runtimes, frameworks, and ecosystems.

What CommandLayer provides

Protocol
  • Canonical verbs for agent actions (e.g. summarize, parse, verify).
  • Strict request & receipt schemas for each verb.
  • Agent identity documents (“agent-cards”) bound to ENS.
  • Discovery via ENS TXT records and x402-style entrypoints.
  • Immutable provenance via checksums and IPFS CIDs.

What CommandLayer is not

CommandLayer does not run agents, host models, or replace orchestration frameworks. It defines:

  • the shared language (verbs),
  • the shared contracts (schemas),
  • the shared identity layer (agent-cards),
  • and the shared routing semantics (ENS + x402).
