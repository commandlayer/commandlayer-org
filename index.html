The canonical verb layer for AI agents.

CommandLayer defines a small, universal set of verbs, JSON Schemas, and agent identity documents that any agent framework can adopt. It does not compete with your models — it standardizes how they talk.

What CommandLayer Provides

  • 10 Commons verbs with strict JSON Schema 2020-12 definitions.
  • Request + receipt schemas for each verb, with trace and x402 envelopes.
  • Agent Cards that bind ENS names, verbs, schemas, and entrypoints.
  • A simple lookup surface for discovery and routing.
