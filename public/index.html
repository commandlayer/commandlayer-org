What you can do right now

Use CommandLayer as the orchestration contract between clients and agents: stable verb names, stable request shapes, and receipts that can be verified.

  1. Pick a verb (start with summarize).
  2. Validate payloads against pinned schemas.
  3. Publish an Agent Card bound to ENS for discovery + routing.
  4. Swap runtimes later without breaking the contract.
// summarize.request
{
"verb": "summarize",
"content": "CommandLayer defines orchestration contracts for agents."
}
// summarize.receipt
