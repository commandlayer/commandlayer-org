Trust pipeline

Parse → Schema → Hash → Signature → Signer

Verification checks are computed from receipt data, signer records, hashes, and signatures.

Schema-valid does not always mean cryptographically valid.

Core checks

  • schema_valid
  • hash_matched
  • signature_valid
  • signer_resolved
  • signer_matched
  • trust_verb
