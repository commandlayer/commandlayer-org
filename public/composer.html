What this demo proves

Commons isn’t just a README — it’s a usable contract you can route through and validate.

  • You pick the verbs and order; the receipt shape stays stable.
  • Each step returns a receipt with proof + trace metadata.
  • Optional chaining turns steps into an actual flow.
  • /verify confirms hash/signature checks per receipt.

Pick verbs, send a flow, inspect receipts + proof checks

This page calls /api/commons-flow (Vercel). The backend calls your live runtime and returns step receipts. The UI can optionally chain outputs and will display /verify results next to each receipt.
