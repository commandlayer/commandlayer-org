What this demo proves

Not a mock. This is your live runtime and your real receipts.

  • Vercel UI → Vercel API → Railway runtime → receipts.
  • Each step is a real POST to /{verb}/v1.0.0.
  • Copy-pasteable curl makes it reproducible.
  • Failures show the raw HTML/JSON error so you can debug fast.
