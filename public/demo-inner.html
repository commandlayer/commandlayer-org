What this surface teaches

Commercial defines economic contract semantics. Runtime executes the flow and may wrap each step response in a structure like { "receipt": ..., "runtime_metadata": ... }. The wrapper is execution context, not the canonical protocol receipt itself.

  • Result / action stays visible for the latest step.
  • Canonical receipt is shown independently of runtime-added metadata.
  • Receipt verification and raw JSON remain inspectable for judges and technical reviewers.
