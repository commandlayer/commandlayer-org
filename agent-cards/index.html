Using Agent Cards

Agent cards are stable, versioned identity documents. They are designed to be fetched and cached by registries, wallets, agent runtimes, and discovery systems. For each verb:

  • Use the schema URLs to validate requests and receipts.
  • Use the agent-card URL to understand ownership and entrypoints.
  • Use ENS TXT + x402 endpoints to route execution.
