Discovery

ENS + x402 for agent discovery and routing.

CommandLayer uses ENS TXT records and x402-style URIs to make verbs, schemas, and entrypoints discoverable in a neutral way.

Example ENS TXT record

cl.verb=summarize
cl.version=1.0.0
cl.entry=x402://summarizeagent.eth/summarize/v1
cl.schema.request=https://commandlayer.org/schemas/v1.0.0/commons/summarize/requests/summarize.request.schema.json
cl.schema.receipt=https://commandlayer.org/schemas/v1.0.0/commons/summarize/receipts/summarize.receipt.schema.json
cl.cid.schemas=<IPFS CID>
cl.checksum=<sha256>

Coordinators, wallets, or other agents can use these records to locate the canonical schema and a reference entrypoint for a verb.

Static lookup JSON

In addition to ENS, a static JSON registry is exposed at /lookup/verbs.json: 

GET https://commandlayer.org/lookup/verbs.json

This allows libraries and tooling to map verbs to schemas and agent-cards without hard-coding individual URLs.
