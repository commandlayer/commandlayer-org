Discovery & Lookup

CommandLayer exposes a simple, static lookup surface that maps verbs to schemas and Agent Cards. Clients can either consume it directly or mirror it internally.

verbs.json

/lookup/verbs.json

Each entry contains:

  • The verb name.
  • Request and receipt schema URLs.
  • The reference Agent Card URL for that verb.
