Governance

Long-lived, neutral semantics for agents.

CommandLayer is intended to be a stable, neutral standard for agent semantics and identity, separate from any single runtime or implementation.

Principles

  • Neutral commons layer (MIT-licensed schemas).
  • Predictable versioning and migration paths.
  • Separation of identity (agent-cards) and execution.
  • Clear distinction between commons and commercial layers.

Evolution

  • New verbs and aliases are coordinated via versioned releases.
  • Existing verbs favor backwards-compatible extensions.
  • Breaking changes require new major versions and clear migration notes.
