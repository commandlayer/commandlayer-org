Getting started

Integrate CommandLayer in 3 steps.

You do not need to change your models, frameworks, or infra. You only need to agree on verbs, schemas, and identifiers.

1. Choose verbs

Start with the Commons verbs: summarize, analyze, fetch, classify, clean, convert, describe, explain, format, parse.

Each verb has a request and receipt schema under /schemas/v1.0.0/commons/<verb>/.

2. Validate requests and receipts

Use the JSON Schemas directly from https://commandlayer.org/schemas/v1.0.0/... or mirror them internally. Validate every incoming request and outgoing receipt.

3. Publish an Agent Card

Create an Agent Card describing your ENS, verbs, schemas, and x402/HTTPS entrypoints. Host it at a stable URL, or mirror the reference cards under /agent-cards/v1.0.0/.
