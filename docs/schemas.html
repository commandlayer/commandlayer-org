Verb Schemas (v1.0.0)

CommandLayer schemas are strict JSON Schema 2020-12 documents covering both requests and receipts for each verb.

Layout

schemas/
└── v1.0.0/
    ├── _shared/
    │   ├── identity.schema.json
    │   ├── receipt.base.schema.json
    │   ├── trace.schema.json
    │   ├── verb.aliases.schema.json
    │   └── x402.schema.json
    └── commons/
        └── <verb>/
            ├── requests/<verb>.request.schema.json
            └── receipts/<verb>.receipt.schema.json

Every schema is addressable via HTTPS and can be mirrored by clients, registries, or internal infra.

Usage

  • Validate incoming requests before execution.
  • Validate receipts before returning them to callers.
  • Use the shared trace and x402 primitives for idempotency and routing.
