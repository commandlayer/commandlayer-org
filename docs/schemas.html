Schemas

Canonical message contracts for agent verbs.

CommandLayer uses strict JSON Schema 2020-12 with deterministic $id and versioned directories for stability and long-term compatibility.

Directory layout (v1.0.0)

schemas/
v1.0.0/
  commons/
    <verb>/
      requests/<verb>.request.schema.json
      receipts/<verb>.receipt.schema.json
  commercial/
    <verb>/
      requests/<verb>.request.schema.json
      receipts/<verb>.receipt.schema.json

The commons layer is a neutral baseline for broadly useful verbs. The commercial layer is for monetized or higher-risk operations where governance and economics may differ.

Versioning principles

  • Semantic versioning per verb and per layer.
  • No breaking changes within a minor version.
  • New features via optional fields and explicit feature-flags.
  • New major versions use new directories (for example v2.0.0/) while preserving old ones.
